# UI/UX Konzept – Birthday Deals App

## Screens

### 1. Onboarding
- Geburtstagsauswahl (Tag + Monat)
- Standortfreigabe
- Kurze Erklärung: "Wir zeigen dir, was du gratis bekommst"

### 2. Hauptseite – Dual View
Oben: Toggle zwischen **Kartenansicht** und **Listenansicht**

#### Kartenansicht (Map)
- Google Maps / Apple Maps Untergrund
- Farbige Pins pro Kategorie
- Tap auf Pin → Mini-Card poppt auf
- Tap auf Card → Detail-View

#### Listenansicht (Cards)
- Horizontaler Kategorie-Filter (Chips oben)
- Scrollbare Deal-Cards
- Card zeigt: Logo, Titel, Entfernung, Kategorie-Icon, Deal-Typ-Badge

### 3. Deal-Detail
- Großes Business-Bild / Logo
- Titel + ausführliche Beschreibung
- Badge: Gültig "Nur am Geburtstag" / "Geburtstagswoche" / "Geburtstagsmonat"
- "Was brauchst du?" → Ausweis / App / Kundenkarte
- Bedingungen (ausklappbar)
- CTA Button: "Deal anzeigen" / "Zur App" / "Coupon öffnen"
- Karte mit Filiale
- "Deal speichern" ♡

### 4. Gespeicherte Deals
- Alle favorisierten Deals
- Sortiert nach: "Heute relevant" zuerst

## Design-Prinzipien

- **Festlich aber clean**: Dezente Geburtstags-Ästhetik, kein Kitsch
- **Kategorie-Farben** als visuelle Orientierung
- **Dark Mode** unterstützt
- **Große Bilder/Logos** — visueller Fokus auf den Deals
- **Badge-System**: "Gratis", "X% Rabatt", "Nur heute" — sofort sichtbar
